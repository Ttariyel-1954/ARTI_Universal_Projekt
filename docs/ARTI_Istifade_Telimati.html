Fəsil 2

Kompüterin açılışı və hazırlıq

Hər səhər kompüterinizi açdıqdan sonra, sistemin işə salınması üçün 4 addım var. Bu ardıcıllıqla getsəniz, bütün sistem 2-3 dəqiqədə tam operativ olacaq.

2.1Sabah — Mac-ı açın

Mac-ı açdıqdan sonra, şifrənizi daxil edin və iş masasına çıxın. Heç bir proqram hələ işləmir — biz onları ardıcıl olaraq işə salacağıq.

2.2Terminal-ı açın

Terminal tətbiqini açmaq üçün:

  • Spotlight (⌘ + Space) açın
  • "Terminal" yazın və Enter basın
  • Qara fon, mətn kursoru görünməlidir — Terminal hazırdır

2.3İlk yoxlama — sistem sağlamlığı

Aşağıdakı əmri kopyalayıb Terminal-da Enter basın:

Terminal — Sistem yoxlanışı 
echo "═══ 1. Günün tarixi: ═══"

Fəsil 4

Tailscale şəbəkəsinin yoxlanışı

Tailscale — sizin Mac ilə bölgə işçilərinin telefonlarını birləşdirən təhlükəsiz virtual şəbəkədir. Hər iki cihaz internet olan yerdə bir-birini görə bilir, amma kənar şəxs görə bilmir.

4.1Tailscale app-ını yoxlayın

Menu bar-a (ekranın yuxarı sağ küncü) baxın. Tailscale icon-u görməlisiniz. Əgər yoxdursa:

  • Applications qovluğunu açın (Finder)
  • Tailscale app-ını tapıb işə salın
  • Menu bar-da icon görünəcək
  • Icon-a klik edib "Connect" olmadığını yoxlayın

4.2Mac-ın Tailscale IP-sini təsdiq edin

Terminal — Tailscale IP 
/Applications/Tailscale.app/Contents/MacOS/Tailscale ip -4

Bu IP həmişə eynidir: 100.95.25.77. Bu ünvanı bölgə işçiləri öz telefonlarına yazırlar.

📝 Bölgə işçilərinin cihazları
Hər bölgə işçisinin telefonunda Tailscale app quraşdırılmalıdır və eyni hesabla (sizin Tailscale hesabınız) giriş etməlidir. Bu birdəfəlik konfiqurasiyadır — bölgəyə işə götürüləndə edilir.
Fəsil 5

Regional app — arxa planda başlat

Regional app — bölgə işçiləri üçündür. Biz onu arxa planda işə salırıq ki, RStudio pəncərəsi dashboard üçün sərbəst qalsın. Terminal bağlansa belə, regional app işləməyə davam edəcək.

5.1Əvvəlki proses varmı — yoxlayın

Terminal — Təmizləmə 
cd ~/Desktop/ARTI_Universal_Platform

# Əvvəlki regional proses varmı?
if [ -f regional.pid ]; then
  OLD_PID=$(cat regional.pid)
  echo "⚠️ Əvvəlki PID: $OLD_PID"
  kill $OLD_PID 2>/dev/null && echo "✅ Dayandırıldı" || echo "ℹ️ Artıq işləmirdi"
  rm regional.pid
fi

# Port 3839-da başqa proses varmı?
PORT_PID=$(lsof -ti:3839)
if [ -n "$PORT_PID" ]; then
  echo "⚠️ Port 3839-da proses: $PORT_PID — öldürülür..."
  kill -9 $PORT_PID 2>/dev/null
  echo "✅ Port boşaldı"
else
  echo "✅ Port 3839 boşdur"
fi

5.2Regional app-ı arxa plana salın

Bu əmr nohup istifadə edir — terminal bağlansa belə app işləyir. Bütün loglar regional.log faylına yazılır.

Terminal — Regional işə sal 
cd ~/Desktop/ARTI_Universal_Platform/domains/construction_management/regional

nohup Rscript -e "options(shiny.port=3839); options(shiny.host='0.0.0.0'); shiny::runApp('.', launch.browser=FALSE)" > ~/Desktop/ARTI_Universal_Platform/regional.log 2>&1 &

REGIONAL_PID=$!
echo $REGIONAL_PID > ~/Desktop/ARTI_Universal_Platform/regional.pid
echo "✅ Regional app PID: $REGIONAL_PID"
echo "✅ PID yazıldı: regional.pid"

# 10 saniyə gözlə paketlər yüklənsin
echo ""
echo "⏳ 10 saniyə gözləyirik..."
sleep 10

# Yoxla
echo ""
echo "═══ Status: ═══"
if lsof -ti:3839 > /dev/null; then
  echo "✅ Regional app işləyir (port 3839)"
  echo ""
  echo "═══ Son log sətirləri: ═══"
  tail -8 ~/Desktop/ARTI_Universal_Platform/regional.log
else
  echo "❌ Regional app başlaya bilmədi"
  echo "═══ Log faylına bax: ═══"
  cat ~/Desktop/ARTI_Universal_Platform/regional.log
fi

🔧 Run App seçimləri

  • Run in Window — ayrı pəncərədə
  • Run in Viewer Pane — RStudio içində (mənim tövsiyəm)

Audit Log hər 30 saniyədə avtomatik yenilənir. Amma əgər siz cədvəldə bir sətri seçmisinizsə (detal panelinə baxırsınızsa), refresh dayanır ki, seçim yoxa çıxmasın. Detal panelini bağladıqdan sonra refresh davam edir.

11.2Regional app-ı dayandırın (əgər istəyirsinizsə)

💭 Regional app-ı dayandırmaq lazımdırmı?
Yox! Regional app-ı gecə də açıq qoya bilərsiniz — çox az resurs istifadə edir (~200 MB RAM). Bölgə işçiləri 24/7 qoşula bilər.

Yalnız bu hallarda dayandırın:
  • Mac-ı tamam söndürəcəksinizsə
  • Regional app-da yeniləmə edirsiniz
  • Port 3839-da başqa iş lazımdır
Terminal — Regional dayandır 
PID_FILE=~/Desktop/ARTI_Universal_Platform/regional.pid

if [ -f "$PID_FILE" ]; then
  PID=$(cat $PID_FILE)
  if ps -p $PID > /dev/null 2>&1; then
    kill $PID
    sleep 2
    if ps -p $PID > /dev/null 2>&1; then
      echo "⚠️ Yumşaq dayanmadı, sərt öldürürük..."
      kill -9 $PID
    fi
    echo "✅ Regional app dayandırıldı (PID: $PID)"
  else
    echo "ℹ️ PID $PID artıq işləmirdi"
  fi
  rm $PID_FILE
else
  echo "ℹ️ PID faylı yoxdur"
  
  # Hər halda port 3839-u yoxla
  PORT_PID=$(lsof -ti:3839)
  if [ -n "$PORT_PID" ]; then
    kill $PORT_PID && echo "✅ Port 3839 proses öldürüldü: $PORT_PID"
  fi
fi

Fəsil 14

Tez-tez verilən suallar

Mac bağlanarsa, regional app-a nə olur?
Regional app dayanır. Telefon işçiləri qoşula bilməyəcəklər. Səhər Mac-ı açanda yenidən Fəsil 5-i icra edin — 3 dəqiqəyə yenidən işləyəcək.
Regional app-ı Mac açıldıqda avtomatik başlaya bilərmi?
Bəli, macOS-un LaunchAgent sistemi ilə bu mümkündür. Amma bu inkişaf etmiş mövzudur — istəsəniz, ayrıca dərs təşkil edərik.
Bir neçə bölgə işçisi eyni anda daxil ola bilərmi?
Bəli! Shiny çoxlu session dəstəkləyir. Yüzlərlə işçi eyni anda qoşula bilər — hər biri öz session-ında işləyir.
Məlumatlar kənar internetə çıxırmı?
Xeyr. Bütün məlumat sizin Mac-da qalır. Tailscale VPN şifrələnmiş kanal yaradır, amma məlumat yalnız sizin cihazlarınız arasında gedir, kənar heç bir serverə yox.
Şifrəni necə dəyişdirim?
Bütün istifadəçi şifrələri bcrypt ilə xeşlənib bazada saxlanır. Dəyişdirmək üçün SQL əmri ilə yeni bcrypt hash yaratmaq lazımdır. Əgər konkret istifadəçinin şifrəsini dəyişmək istəyirsinizsə, bir növbəti dərsdə edə bilərik.
Foto həqiqətən saxlanır?
Bəli. Bütün fotolar ~/Desktop/ARTI_Universal_Platform/uploads/ qovluğuna saxlanır. Alt qovluqlar: inspections, complaints, daily_reports, material_movements. Bazada isə construction.attachments cədvəlində metadata qeyd olunur.
Regional app-da təzə feature əlavə etmək istəsəm?
regional/app.R faylı dəyişdirilir. Sonra: əvvəlki regional app-ı dayandırırsınız (Fəsil 11.2), yenidən başladırsınız (Fəsil 5.2). Yeni kod aktivləşir, telefon işçiləri avtomatik yeniliyini görəcəklər.
Backup necə edim?
Ən vacibi PostgreSQL bazasıdır. Gündəlik avtomatik backup üçün pg_dump arti_construction > backup_$(date +%Y%m%d).sql əmri istifadə olunur. Həmçinin uploads/ qovluğunu da Time Machine-ə daxil edin.
Yeni bölgə işçisi əlavə etmək?
Yeni işçi üçün audit.users cədvəlinə INSERT lazımdır (username, password_hash, role='regional_worker', region_id). Bu admin rolunun səlahiyyətidir — dashboard-da "İstifadəçilər" tab əlavə edilsə, bu GUI-dən edilə bilər. Plana qoyulub.
